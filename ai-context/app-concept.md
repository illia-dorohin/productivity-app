# Productivity App — Концепт

## Основна ідея

Персональний додаток для щоденного трекінгу продуктивності та самооцінки.
Щодня користувач ставить оцінки від -2 до +2 по різних показниках,
які автоматично агрегуються в тижневі та місячні звіти через ієрархічну систему груп.

Додатково є розділи для планування задач та ведення нотаток.

- Авторизація через Google
- Адаптивний дизайн (мобільна версія важлива)
- Інтерфейс англійською мовою

---

## Ієрархія оцінок

Всі пункти оцінюються від -2 до +2. Повторне натискання на вибрану оцінку скидає вибір.
Групи рахуються як зважене середнє по дочірніх елементах (вага — налаштовується, за замовчуванням 1).
Загальна оцінка — зважене середнє по верхньорівневих групах.

Пропущені пункти/дні — ігноруються при розрахунку (середнє лише по заповнених).

```
Overall Score (зважене середнє по верхніх групах, ваги = 1 за замовчуванням)
│
├── Productivity (зважене середнє: Work + Additional Activities)
│   ├── Work (один пункт, поки без підгруп)
│   └── Additional Activities (середнє по всіх доп. активностях)
│       ├── Drone Factory
│       ├── Glass
│       ├── Creativity
│       └── Other Activities
│
├── Brain Rot (середнє: Games + Other Destructive)
│   ├── Computer Games (+2 = не грав)
│   └── Other Destructive Activities (+2 = не витрачав час)
│
├── Health
│   └── Sport (поки один пункт, можна додавати — чекапи тощо)
│
├── Mental Health (один пункт — загальна оцінка ментального здоров'я)
│
└── Relationships
    ├── Communication with Partner
    └── Actions towards Partner
```

## Конфігурація показників (Admin UI)

Окремий розділ в додатку для управління ієрархією:
- Додавати нові показники та групи
- Видаляти/деактивовувати (без втрати старих даних)
- Змінювати ієрархію (переміщати пункти між групами)
- Налаштовувати ваги для розрахунку

---

## Розділ: Плани / Задачі

Гнучка система без обов'язкових полів. Можна використовувати як простий чекліст або як детальний task manager.

Кожна задача має:
- **Назва** (обов'язкове)
- **Чекбокс** — швидке перемикання done/not done (під капотом змінює статус)
- **Статус** — not started / in progress / done
- **Прогрес** — процент виконання (опціонально)
- **Дедлайн** — дата (опціонально)
- **Пріоритет** — high / medium / low (опціонально)
- **Категорія / Теги** (опціонально)
- **Опис / Нотатки** (опціонально)

Без підзадач.

---

## Розділ: Нотатки

Окремий розділ нотаток (не прив'язаний до сторінки конкретного дня):
- Текст + дата створення
- Категорія або тег (опціонально)
- Декілька нотаток на один день — можна
- Простий текст (без rich text / markdown)

## Звіти та візуалізація

### Тижневий звіт
- Середня оцінка по кожному показнику
- Середня оцінка по кожній групі
- Overall Score за тиждень

### Місячний звіт
- Те саме що тижневий, але за місяць
- Тренди (як змінювались оцінки протягом місяця)

### Графіки
- **Лінійний графік трендів** — зміна оцінок (overall + по групах) з часом (дні/тижні)
- **Radar chart** — порівняння верхньорівневих груп за обраний період (наочне відображення балансу)
- **Heatmap календар** — загальна оцінка по днях, як на GitHub (швидкий візуальний огляд)

## Сторінки додатку (UI)

1. **Login** — авторизація через Google
2. **Daily Rating** — головна сторінка, оцінки за сьогодні (або обраний день)
3. **Reports** — тижневі та місячні звіти з графіками
4. **Tasks** — плани та задачі
5. **Notes** — нотатки
6. **Settings / Admin** — управління показниками, вагами, ієрархією
